角色设定：
你是一位精通 JavaFX 和 CSS 的高级 UI/UX 设计师。请帮我重构现有的 JavaFX 代码，将界面升级为现代极简风格（类似 VS Code 或 Notion 的风格）。

当前问题：
目前的界面布局拥挤，组件之间没有间距，颜色单调（全是系统默认灰），字体较小且不可读性差。

设计要求：

整体配色： 使用深色模式（Dark Theme）。背景色使用深灰（如 #1e1e1e），文字使用浅灰或白色（#d4d4d4）。
布局优化：
使用 VBox 作为主容器，设置全局内边距（Padding）为 20px，子元素间距（Spacing）为 15px。
顶部导航栏： 将 TabPane 样式改为无边框设计，选中的 Tab 高亮显示（例如底部有一条蓝色横线），未选中的 Tab 文字变暗。
工具栏区域： “Delimiter”输入框和按钮组（Join, Split...）应该放在同一行（HBox），并且垂直居中对齐。按钮需要增加圆角（Radius 5px）和内边距，鼠标悬停时要有变色反馈。
输入输出区： TextArea 需要去除默认的立体边框，改为扁平化设计。给它们加上圆角，并设置内部 Padding 为 10px，防止文字紧贴边缘。
字体优化： 全局字体大小调整为 14px，输入框和代码区域使用等宽字体（Monospaced），如 'Consolas' 或 'JetBrains Mono'。
交互细节： 所有的输入框和文本域在获得焦点（Focus）时，边框颜色变为强调色（如蓝色 #007acc）。
